## ğŸ” 6. HTTPS vs HTTP (What Wireshark Can See)

### **HTTP (Insecure)**
Wireshark can read:
- Username  
- Password  
- Full form data  
- URL paths  
- Cookies  
- All request details  

### **HTTPS (Secure with TLS)**
Wireshark only sees:
- IP addresses  
- TLS handshake  
- Encrypted traffic  

Wireshark **cannot** see:
- Username  
- Password  
- Form content  
- Query parameters  
- Page text  

TLS encrypts the entire HTTP payload.

---

## ğŸ›¡ï¸ 7. What TLS Is (Simple Explanation)

TLS = **Transport Layer Security**  
It encrypts everything between client and server so attackers cannot read the data.

TLS protects:
- Login forms  
- Session cookies  
- Personal data  
- Website content  

Without TLS, all data is exposed.
